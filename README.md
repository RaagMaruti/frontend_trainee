# Frontend - Training

This repository contains the projects and assignments completed during the internship program. The structure and purpose of each folder are detailed below.

## Folder Structure

### 01.html
Contains examples and assignments related to HTML, focusing on the following:
1. Elements and Attributes: p, div, span, text styling, id, class
2. Lists and Tables
3. Links and Images
4. Forms: inputs, labels, their attributes
5. Semantics and Layout
6. Iframe vs. embed vs. object
7. Audio, Video
8. Graphics: SVG, Canvas, Favicon
9. APIs


### 02.css
Contains examples and assignments related to CSS, focusing on the following:
1. CSS Basics: fonts, positions, z-index
2. Colors: formats, gradient, opacity, shadows
3. Animations: 2d, 3d
4. Peudo: classes, elements
5. Variables
6. Box Model: margin, border, padding, content
7. RWD: flexbox, grid, media queries
